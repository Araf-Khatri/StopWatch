**What Buttons do?**
```
Start: Starts Timer.
Pause: Pause Timer.
Resume: Starts paused timer.
Lap: Create a lap that shows Lap number, Time when the lap button was clicked & Time took to depending on previous lap.  
Reset: clear all the Laps & Time
```
